# Telco Customer Revenue & Churn Analysis

## Project Overview
This project analyzes customer churn and revenue drivers for a telecom business using SQL-based data analysis and financial metrics. The goal is to identify key factors driving customer churn, quantify revenue exposure, and provide insights to improve customer retention and revenue stability.

## Dataset
Telco Customer Churn dataset (~7,000 customer records)

## Tools
SQL  
Excel  
Financial KPI Analysis  
Data Visualization
